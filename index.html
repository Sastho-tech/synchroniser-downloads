Installation Guide

Step-by-step instructions for your platform

Installing on macOS

1

Download the DMG file

Choose the appropriate version for your Mac (Apple Silicon or Intel)

Not sure which Mac you have?

Click Apple menu → About This Mac
• Shows "Apple M1/M2/M3" → Download Apple Silicon
• Shows "Intel Core i5/i7/i9" → Download Intel

2

Install the Application

Open the DMG file and drag Synchroniser to your Applications folder

Important: macOS will block the app initially for security. Follow the next step.

3

Remove macOS Quarantine (Required)

Open Terminal and run this command:

xattr -dr com.apple.quarantine /Applications/Synchroniser.app

Critical: Without this step, the app cannot auto-launch Docker Desktop!

4

Launch Synchroniser

Option 1: GUI - Go to Applications folder and double-click Synchroniser

If you see a security warning, right-click the app → Open → Open again

Option 2: Terminal - Run from command line: (Recommended)

/Applications/Synchroniser.app/Contents/MacOS/Synchroniser
5

Install Docker Desktop (if needed)

If you don't have Docker Desktop, the app will guide you to install it

Download Docker Desktop for Mac

Quick Reference

User Data:
~/Library/Application Support/Synchroniser

Installing on Windows

1

Download the Installer

Download Synchroniser-Setup-1.0.3.exe from the download section

2

Run the Installer

Double-click the .exe file to start installation

Windows SmartScreen Warning: Click "More info" → "Run anyway"

3

Follow Installation Wizard

Accept the license agreement and choose installation location (default: C:\Program Files\Synchroniser)

4

Launch Synchroniser

The app will launch automatically after installation, or find it in Start Menu

5

Install Docker Desktop (if needed)

Download and install Docker Desktop for Windows

Download Docker Desktop for Windows

Note: You may need to restart your computer after installing Docker Desktop

Quick Reference

App Location:
C:\Program Files\Synchroniser
User Data:
%APPDATA%\Synchroniser

Installing on Linux

1

Download the AppImage

Download Synchroniser-1.0.3.AppImage from the download section

2

Make it Executable

Open terminal and navigate to Downloads folder, then run:

chmod +x Synchroniser-*.AppImage
3

Run Synchroniser

Execute the AppImage:

./Synchroniser-*.AppImage

Or double-click the file in your file manager

4

Install FUSE (if needed)

If AppImage won't run, install FUSE:

# Ubuntu/Debian
sudo apt-get install fuse libfuse2

# Fedora
sudo dnf install fuse fuse-libs

# Arch
sudo pacman -S fuse2
5

Install Docker (if needed)

Install Docker Engine on your Linux distribution:

# Ubuntu/Debian - Quick install
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# Add user to docker group
sudo usermod -aG docker $USER

# Log out and back in for group changes

Frequently Asked Questions

Do I need Docker Desktop installed?

Yes, Docker Desktop is required for Synchroniser to work. The app will help guide you through the installation process if Docker isn't installed.

Will this work on Apple Silicon (M1/M2/M3)?

Yes! We provide native Apple Silicon builds for optimal performance on M1, M2, and M3 Macs.

Is Synchroniser free to use?

Yes, Synchroniser is completely free and open source. You can use it for personal or commercial purposes.

How do I update Synchroniser?

The app will notify you when updates are available. Simply download the latest version from this page and reinstall. Your data and settings will be preserved.
